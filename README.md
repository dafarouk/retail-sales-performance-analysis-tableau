# Retail Sales Performance Analysis (Tableau)

## Project Overview
This project analyzes retail store performance, customer behavior, and profitability using Tableau dashboards and 20+ business KPIs.

## Objectives
- Analyze sales and profit trends
- Evaluate store efficiency and performance
- Identify top customers and loyalty patterns
- Assess revenue quality and profitability

## Tools Used
- Tableau
- CSV datasets
- Calculated KPIs:
  - Average Basket Value
  - Profit Margin %
  - Store Efficiency Score
  - Customer Loyalty Indicator

## Dashboards
- Sales Overview
- Store Performance
- Customer Analysis
- Revenue & Profit Quality

## Files Included
- Tableau packaged workbook (.twbx)
- Dataset CSV files
- Presentation PDF
